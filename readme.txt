# AI_Project1
Uniformed search for AI project 1

Rece Coffin (rcoffin)

Written in Java

Developed, compiled, and ran on Linux Mint

The code is structured as a single Java class with 2 internal classes
one to represent a node, and one to represent an edge in the graph.

zip file includes an already packaged .jar file. To execute the file
type the following command into any machine with java installed.

java -jar find_route.jar <arg1> <arg2> <arg3>
